Copilot Proxy — Remote Setup

Configure Claude Code on a remote device to use this proxy.

On host machine

Step 1: Start the setup approval server

.\copilotproxy.ps1 setup-claude-remote

This starts an interactive approval server. When a remote device connects, it will show the hostname/IP and prompt you to approve (y/N). Keep it running.

On remote device

Step 2: Download the setup script

Requires approval from the host machine (Step 1). The host will see your hostname/IP.

On remote device

Step 3: Review the script

cat claude-copilot-proxy.sh

Always review scripts before executing them. Verify the proxy URL and token look correct.

On remote device

Step 4: Run the setup script

sh claude-copilot-proxy.sh

Handles everything: installs devtunnel (if needed), logs in, connects in background with auto-reconnect, and configures Claude Code.
Other commands: sh claude-copilot-proxy.sh reconnect, sh claude-copilot-proxy.sh stop, sh claude-copilot-proxy.sh disable
